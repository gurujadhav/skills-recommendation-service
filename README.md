# Skills Recommendation Service (Flask + MySQL)

## 🚀 Overview
A REST API that manages users and their skills, and provides recommendations.

## 🛠️ Tech Stack
- Python (Flask)
- MySQL
- Pytest (Testing)

## 📂 Endpoints
### 1. Add User
`POST /user`
```json
{
  "name": "Alice",
  "skills": ["Python", "Java"]
}
